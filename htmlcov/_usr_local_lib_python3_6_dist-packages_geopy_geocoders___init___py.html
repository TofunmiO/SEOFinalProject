1""" 

2Each geolocation service you might use, such as Google Maps, Bing Maps, or 

3Nominatim, has its own class in ``geopy.geocoders`` abstracting the service's 

4API. Geocoders each define at least a ``geocode`` method, for resolving a 

5location from a string, and may define a ``reverse`` method, which resolves a 

6pair of coordinates to an address. Each Geocoder accepts any credentials 

7or settings needed to interact with its service, e.g., an API key or 

8locale, during its initialization. 

9 

10To geolocate a query to an address and coordinates: 

11 

12 >>> from geopy.geocoders import Nominatim 

13 >>> geolocator = Nominatim(user_agent="specify_your_app_name_here") 

14 >>> location = geolocator.geocode("175 5th Avenue NYC") 

15 >>> print(location.address) 

16 Flatiron Building, 175, 5th Avenue, Flatiron, New York, NYC, New York, ... 

17 >>> print((location.latitude, location.longitude)) 

18 (40.7410861, -73.9896297241625) 

19 >>> print(location.raw) 

20 {'place_id': '9167009604', 'type': 'attraction', ...} 

21 

22 

23To find the address corresponding to a set of coordinates: 

24 

25 >>> from geopy.geocoders import Nominatim 

26 >>> geolocator = Nominatim(user_agent="specify_your_app_name_here") 

27 >>> location = geolocator.reverse("52.509669, 13.376294") 

28 >>> print(location.address) 

29 Potsdamer Platz, Mitte, Berlin, 10117, Deutschland, European Union 

30 >>> print((location.latitude, location.longitude)) 

31 (52.5094982, 13.3765983) 

32 >>> print(location.raw) 

33 {'place_id': '654513', 'osm_type': 'node', ...} 

34 

35Locators' ``geocode`` and ``reverse`` methods require the argument ``query``, 

36and also accept at least the argument ``exactly_one``, which is ``True`` by 

37default. 

38Geocoders may have additional attributes, e.g., Bing accepts ``user_location``, 

39the effect of which is to bias results near that location. ``geocode`` 

40and ``reverse`` methods may return three types of values: 

41 

42- When there are no results found, returns ``None``. 

43 

44- When the method's ``exactly_one`` argument is ``True`` and at least one 

45 result is found, returns a :class:`geopy.location.Location` object, which 

46 can be iterated over as: 

47 

48 ``(address<String>, (latitude<Float>, longitude<Float>))`` 

49 

50 Or can be accessed as ``location.address``, ``location.latitude``, 

51 ``location.longitude``, ``location.altitude``, and ``location.raw``. The 

52 last contains the full geocoder's response for this result. 

53 

54- When ``exactly_one`` is ``False``, and there is at least one result, returns a 

55 list of :class:`geopy.location.Location` objects, as above: 

56 

57 ``[location, [...]]`` 

58 

59If a service is unavailable or otherwise returns a non-OK response, or doesn't 

60receive a response in the allotted timeout, you will receive one of the 

61`Exceptions`_ detailed below. 

62 

63.. _specifying_parameters_once: 

64 

65Specifying Parameters Once 

66~~~~~~~~~~~~~~~~~~~~~~~~~~ 

67 

68Geocoding methods accept a lot of different parameters, and you would 

69probably want to specify some of them just once and not care about them 

70later. 

71 

72This is easy to achieve with Python's :func:`functools.partial`:: 

73 

74 >>> from functools import partial 

75 >>> from geopy.geocoders import Nominatim 

76 

77 >>> geolocator = Nominatim(user_agent="specify_your_app_name_here") 

78 

79 >>> geocode = partial(geolocator.geocode, language="es") 

80 >>> print(geocode("london")) 

81 Londres, Greater London, Inglaterra, SW1A 2DX, Gran Bretaña 

82 >>> print(geocode("paris")) 

83 París, Isla de Francia, Francia metropolitana, Francia 

84 >>> print(geocode("paris", language="en")) 

85 Paris, Ile-de-France, Metropolitan France, France 

86 

87 >>> reverse = partial(geolocator.reverse, language="es") 

88 >>> print(reverse("52.509669, 13.376294")) 

89 Steinecke, Potsdamer Platz, Tiergarten, Mitte, 10785, Alemania 

90 

91If you need to modify the query, you can also use a one-liner with lambda. 

92For example, if you only need to geocode locations in `Cleveland, Ohio`, 

93you could do:: 

94 

95 >>> geocode = lambda query: geolocator.geocode("%s, Cleveland OH" % query) 

96 >>> print(geocode("11111 Euclid Ave")) 

97 Thwing Center, Euclid Avenue, Magnolia-Wade Park Historic District, 

98 University Circle, Cleveland, Cuyahoga County, Ohio, 44106, United States 

99 of America 

100 

101That lambda doesn't accept kwargs. If you need them, you could do:: 

102 

103 >>> _geocode = partial(geolocator.geocode, language="es") 

104 >>> geocode = lambda query, **kw: _geocode("%s, Cleveland OH" % query, **kw) 

105 >>> print(geocode("11111 Euclid Ave")) 

106 Thwing Center, Euclid Avenue, Magnolia-Wade Park Historic District, 

107 University Circle, Cleveland, Cuyahoga County, Ohio, 44106, Estados Unidos 

108 >>> print(geocode("11111 Euclid Ave", language="en")) 

109 Thwing Center, Euclid Avenue, Magnolia-Wade Park Historic District, 

110 University Circle, Cleveland, Cuyahoga County, Ohio, 44106, United States 

111 of America

113Geopy Is Not a Service 

114~~~~~~~~~~~~~~~~~~~~~~ 

115 

116Geocoding is provided by a number of different services, which are not 

117affiliated with geopy in any way. These services provide APIs, which anyone 

118could implement, and geopy is just a library which provides these 

119implementations for many different services in a single package. 

120 

121.. image:: ./_static/geopy_and_geocoding_services.svg 

122 :target: ./_static/geopy_and_geocoding_services.svg 

123 

124Therefore: 

125 

1261. Different services have different Terms of Use, quotas, pricing, 

127 geodatabases and so on. For example, :class:`.Nominatim` 

128 is free, but provides low request limits. If you need to make more queries, 

129 consider using another (probably paid) service, such as 

130 :class:`.OpenMapQuest` or :class:`.PickPoint` 

131 (these two are commercial providers of Nominatim, so they should 

132 have the same data and APIs). Or, if you are ready to wait, you can try 

133 :mod:`geopy.extra.rate_limiter`. 

134 

1352. geopy cannot be responsible for the geocoding services' databases. 

136 If you have issues with some queries which the service cannot fulfill, 

137 it should be directed to that service's support team. 

138 

1393. geopy cannot be responsible for any networking issues between your computer 

140 and the geocoding service. 

141 

142If you face any problem with your current geocoding service provider, you can 

143always try a different one.

145.. _async_mode: 

146 

147Async Mode 

148~~~~~~~~~~ 

149 

150By default geopy geocoders are synchronous (i.e. they use an Adapter 

151based on :class:`.BaseSyncAdapter`). 

152 

153All geocoders can be used with asyncio by simply switching to an 

154Adapter based on :class:`.BaseAsyncAdapter` (like :class:`.AioHTTPAdapter`). 

155 

156Example:: 

157 

158 from geopy.adapters import AioHTTPAdapter 

159 from geopy.geocoders import Nominatim 

160 

161 async with Nominatim( 

162 user_agent="specify_your_app_name_here", 

163 adapter_factory=AioHTTPAdapter, 

164 ) as geolocator: 

165 location = await geolocator.geocode("175 5th Avenue NYC") 

166 print(location.address) 

167 

168Basically the usage is the same as in synchronous mode, except that 

169all geocoder calls should be used with ``await``, and the geocoder 

170instance should be created by ``async with``. The context manager is optional, 

171however, it is strongly advised to use it to avoid resources leaks. 

172

287def get_geocoder_for_service(service): 

288 """ 

289 For the service provided, try to return a geocoder class. 

290 

291 >>> from geopy.geocoders import get_geocoder_for_service 

292 >>> get_geocoder_for_service("nominatim") 

293 geopy.geocoders.nominatim.Nominatim 

294 

295 If the string given is not recognized, a 

296 :class:`geopy.exc.GeocoderNotFound` exception is raised. 

297 

298 Given that almost all of the geocoders provide the ``geocode`` 

299 method it could be used to make basic queries based entirely 

300 on user input:: 

301 

302 from geopy.geocoders import get_geocoder_for_service 

303 

304 def geocode(geocoder, config, query): 

305 cls = get_geocoder_for_service(geocoder) 

306 geolocator = cls(**config) 

307 location = geolocator.geocode(query) 

308 return location.address 

309 

310 >>> geocode("nominatim", dict(user_agent="specify_your_app_name_here"), \ 

311"london") 

312 'London, Greater London, England, SW1A 2DX, United Kingdom' 

313 >>> geocode("photon", dict(), "london") 

314 'London, SW1A 2DX, London, England, United Kingdom' 

315 

316 """ 

317 try: 

318 return SERVICE_TO_GEOCODER[service.lower()] 

319 except KeyError: 

320 raise GeocoderNotFound( 

321 "Unknown geocoder '%s'; options are: %s" % 

322 (service, SERVICE_TO_GEOCODER.keys()) 

323 )
